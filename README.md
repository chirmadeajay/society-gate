# 🏢 Society Gate — AI-Powered Visitor Management System

A full-stack production-grade visitor management app for residential societies.

## 📱 Features
- Guard adds visitor at gate
- Resident gets WhatsApp notification instantly
- Resident taps Approve ✅ or Reject ❌
- Real-time status update
- Complete visitor history log

## 🚀 Tech Stack
- **Backend:** FastAPI + PostgreSQL + Redis
- **Frontend:** React (Admin Dashboard)
- **Mobile:** React Native (Expo Go)
- **Auth:** JWT
- **Notifications:** Twilio WhatsApp API
- **Deploy:** Docker + Railway + Vercel
